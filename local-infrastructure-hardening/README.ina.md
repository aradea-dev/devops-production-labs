# Local Infrastructure Hardening - Labs

Repositori ini berisi dokumentasi dan hasil latihan otomasi serta pengerasan keamanan infrastruktur lokal (WSL2).

## Fitur Keamanan yang Diterapkan:
- **Hari 1:** Fondasi OS (User Non-root) & Strict Mode Bash (`set -euo pipefail`).
- **Hari 2:** Otomasi Latar Belakang (`cron`) & Error Handling (`trap`).
- **Hari 3:** Edge Security Hardening (Nginx Server Tokens Off, Rate Limiting 5r/s, Security Headers).
- **Hari 4:** HTTPS Enkripsi Lokal menggunakan `mkcert` pada Port 443.
